# OOP with Python — Full Course Repository

A complete 16-week Object-Oriented Programming course in Python.
Includes weekly exercises, assignments, and revision notes.

---

## Course outline

| Week | Topic | Type |
|------|-------|------|
| Week 1  | Classes, Objects, and the self Keyword         | Topic    |
| Week 2  | Encapsulation and Property Decorators           | Topic    |
| Week 3  | Magic Methods and Operator Overloading          | Topic    |
| Week 4  | Inheritance, Overriding, and Composition        | Topic    |
| Week 5  | Polymorphism and Abstract Base Classes          | Topic    |
| Week 6  | Static/Class Methods and Custom Decorators      | Topic    |
| Week 7  | Revision                                        | Revision |
| Week 8  | Mid-term Exam                                   | Exam     |
| Week 9  | Dataclasses and Type Hinting                    | Topic    |
| Week 10 | Custom Exceptions and Error Handling            | Topic    |
| Week 11 | Iterators, Generators, and Context Managers     | Topic    |
| Week 12 | Object Serialization (JSON and Pickle)          | Topic    |
| Week 13 | Concurrent Programming (Threads & Multiprocessing) | Topic |
| Week 14 | Design Patterns and Unit Testing                | Topic    |
| Week 15 | Revision                                        | Revision |
| Week 16 | Final Exam                                      | Exam     |

---

> Weeks 8 and 16 are exams — no code folders needed.
